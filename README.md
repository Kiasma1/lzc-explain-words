# ljg-explain-words

A Codex skill for deep English word mastery. It generates museum-quality HTML word cards that explain one or more English words through:

- core semantic image
- etymology and cognates
- nuance comparisons
- Mermaid semantic topology
- a bilingual epiphany line

## Contents

- `SKILL.md` — skill instructions and usage contract
- `assets/word_card.html` — adaptive HTML template
- `scripts/render_word_cards.py` — renderer for single or multiple structured entries

## Features

- supports **one word or multiple words** from user input
- adaptive handling for **long titles**
- adaptive handling for **long phonetics**
- Mermaid node **auto-wrapping**
- Mermaid topology **density-aware layout**
- multi-word render mode generates an **index page**

## Render from structured JSON

```bash
python3 scripts/render_word_cards.py \
  --input /path/to/words.json \
  --output-dir /path/to/output
```

### Input shape

Single entry:

```json
{
  "word": "Excerpt",
  "phonetic": "ˈek.sɝːpt",
  "definition_deep": "<p>...</p>",
  "etymology": "<p>...</p>",
  "nuance_text": "<ul class=\"nuance-list\"><li class=\"nuance-item\">...</li></ul>",
  "example_sentence": "She quoted a brief excerpt.",
  "epiphany": "An excerpt is a chosen window. 摘录是被选择出来的一扇窗。",
  "mermaid_code": "graph TD\\nA[whole text] --> B[selected passage]"
}
```

Multiple entries:

```json
[
  {
    "word": "Excerpt",
    "phonetic": "ˈek.sɝːpt",
    "definition_deep": "<p>...</p>",
    "etymology": "<p>...</p>",
    "nuance_text": "<ul class=\"nuance-list\"><li class=\"nuance-item\">...</li></ul>",
    "example_sentence": "She quoted a brief excerpt.",
    "epiphany": "An excerpt is a chosen window. 摘录是被选择出来的一扇窗。",
    "mermaid_code": "graph TD\\nA[whole text] --> B[selected passage]"
  },
  {
    "word": "Serendipity",
    "phonetic": "ˌser.ənˈdɪp.ə.ti",
    "definition_deep": "<p>...</p>",
    "etymology": "<p>...</p>",
    "nuance_text": "<ul class=\"nuance-list\"><li class=\"nuance-item\">...</li></ul>",
    "example_sentence": "Their meeting was pure serendipity.",
    "epiphany": "Serendipity is chance meeting a prepared soul. 机缘是偶然遇见了有准备的灵魂。",
    "mermaid_code": "graph TD\\nA[searching] --> B[finding]"
  }
]
```

## Output

- one entry → one `word_card_<slug>.html`
- multiple entries → multiple HTML cards + `word_cards_index.html`

## Notes

- The card template currently loads Mermaid from CDN.
- Structured word analysis content is expected to be generated by the calling agent/workflow before rendering.
